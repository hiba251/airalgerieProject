# Air Algérie Web Scraping Project

## Objectif
Ce projet a pour objectif d’extraire automatiquement les offres promotionnelles d’Air Algérie à partir de leur site web et de transformer ces données dynamiques en informations structurées et exploitables.

## Technologies utilisées
- Python
- Selenium
- Pandas
- Export CSV / JSON

## Fonctionnalités
- Collecte automatisée des données
- Nettoyage et structuration des informations
- Génération de fichiers exploitables pour l’analyse
